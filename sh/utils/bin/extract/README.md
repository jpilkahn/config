# Delegating meta-utility for extracting compressed archives.

| file extension | format  | cmd delegation |
|----------------|---------|----------------|
| `.7z`          | 7zip    | `7z x`         |
| `.bz2`         | bz2     | `bunzip2`      |
| `.gz`          | gz      | `gunzip`       |
| `.rar`         | rar     | `unrar`        |
| `.tar`         | tar     | `tar xf`       |
| `.tar.bz2`     | tar.bz2 | `tar xjf`      |
| `.tbz2`        |         | `tar xjf`      |
| `.tar.gz`      | tar.gz  | `tar xzf`      |
| `.tgz`         |         | `tar xzf`      |
| `.zip`         | zip     | `unzip`        |
| `.Z`           | Z       | `uncompress`   |
